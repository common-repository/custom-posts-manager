== Description ==

**Custom Posts Manager** enables you to easily clone or duplicate your WordPress content, including standard posts, pages, and any custom post types registered on your site. == Installation ==

1. Upload the `custom-posts-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Navigate to any post, page, or custom post type list view to use the Clone feature.

== Frequently Asked Questions ==

= How do I clone a post, page, or custom post type? =

In the list view of posts, pages, or custom post types, hover over the item you wish to clone and click the Clone link. The cloned item will appear as a draft copy in the same list.

= Does the plugin support custom post types? =

Yes, you can clone any registered custom post type on your site. Make sure the custom post type is enabled in the plugin settings under CP Manager if it is not appearing in the list view.
